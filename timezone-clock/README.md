<!-- Приложение для отображения времени в нескольких часовых поясах

Разработано в рамках тестового задания на позицию Junior Frontend разработчика

Интерфейсом приложения предусмотрена возможность изменения количества отображаемых на
странице часов от 1 до 10.

Список часовых поясов получен путем выполнения сетевого запроса и загрузки
данных из файла локального файла «timezones.json».

Для клонирования репозитория выполните команду: git clone https://github.com/Artcode-x/sky-fitness-pro При разработке рекомендуется использовать пакетного менеджера npm. Перед открытием проекта необходимо установить библиотеку. npm install Запуск локального сервера с проектом осуществляется командой: npm start Проект будет запущен по адресу: http://localhost:3000

Стэк библиотеки проекта: React Redux Axios -->
